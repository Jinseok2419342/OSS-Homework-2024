#### 수정 작업 취소
`$ git restore` 
스테이징에서 워킹 트리로 
`$ git restore --staged`
커밋에서 스테이지 에리아로

작업 영역과 스테이지 영역 파일 수정 취소
- 옵션--staged : index 파일unstage(index를최근커밋으로수정[복사])
